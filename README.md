# 🧮 Calculadora Pro de Métodos Numéricos

¡Bienvenido! Esta es una aplicación web interactiva diseñada para resolver ecuaciones utilizando los métodos de **Newton-Raphson** y **Series de Taylor**. Proyecto desarrollado para la clase de Métodos Numéricos.

 **[¡Prueba la aplicación aquí en vivo!](https://hugo4294.github.io/pagina-web-educativa-/)**

---

##  Características Principales
* **Cálculo Automático:** Generación de matrices de convergencia detalladas con precisión decimal configurable.
* **Gráficas en Tiempo Real:** Visualización interactiva de la función seleccionada.
* **Interfaz Adaptable (Responsive):** Diseño moderno optimizado para computadoras y teléfonos celulares.
* **Menú de Ejemplos:** Acceso rápido a funciones predefinidas para pruebas rápidas.

##  Tecnologías Utilizadas
* **HTML5 & CSS3:** Estructura limpia y diseño personalizado con tarjetas flotantes.
* **JavaScript (ES6):** Lógica del algoritmo de aproximación de raíces.
* **Math.js:** Para el análisis y procesamiento estricto de funciones matemáticas.
* **Chart.js:** Renderizado dinámico de las gráficas de las funciones.
