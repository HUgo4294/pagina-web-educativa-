📋 Algoritmo de Newton-Raphson

Paso 1: Encuentra puntos \(a\) y \(b\) tales que \(a < b\) y \(f(a) \cdot f(b) < 0\).
Paso 2: Toma el intervalo \([a, b]\) y calcula el valor inicial \(x_0 = \frac{a+b}{2}\).
Paso 3: Halla \(f(x_0)\) y \(f'(x_0)\) para calcular la aproximación: \(x_1 = x_0 - \frac{f(x_0)}{f'(x_0)}\).
Paso 4: Si \(f(x_1) = 0\), entonces \(x_1\) es una raíz exacta; de lo contrario, \(x_0 = x_1\).
Paso 5: Repita los pasos 3 y 4 hasta que \(f(x_i) = 0\) o \(|f(x_i)| \le \text{Tolerancia}\).

Ejemplo 1: Función Polinómica Cúbica

f(x) = x^3 - x - 1 = 0

Intervalo [1, 2]. Cambio de signo garantizado. Usando x0 = 1.5 converge rápidamente a ≈ 1.3247.

Ejemplo 2: Parábola (Raíz Exacta)

f(x) = x^2 - 4 = 0

Derivada 2x. Partiendo de x0 = 3, intersecta el eje horizontal directo en x = 2.

Guía Teórica

Newton-Raphson

Aproximación lineal mediante rectas tangentes sucesivas.

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

N-R Modificado

Diseñado para conservar la convergencia rápida ante raíces múltiples.

$$x_{n+1} = x_n - \frac{f(x_n)f'(x_n)}{[f'(x_n)]^2 - f(x_n)f''(x_n)}$$
