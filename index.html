Guía Algorítmica Rápida

1. Define la función continua f(x) y estima un punto inicial x₀.
2. Evalúa la tasa de cambio mediante su primera derivada f'(x).
3. Aplica la fórmula iterativa para aproximar progresivamente el punto de raíz.

Selecciona un caso de estudio:

Polinómica Estándar

f(x) = x³ - x - 2 | x₀ = 1.5
Ideal para probar la convergencia cuadrática clásica sin complicaciones iniciales.

Trascendente (Trigonométrica)

f(x) = cos(x) - x | x₀ = 0.5
Punto de cruce clásico de funciones trigonométricas con comportamiento estable.

Raíz Múltiple (Caso Crítico)

f(x) = x³ - 3x² + 3x - 1 | x₀ = 1.5
Tiene raíz triple en x=1. Se recomienda usar el Método Modificado para no perder velocidad.

Guía Teórica

Newton-Raphson (Raíces simples)

Aproximación lineal mediante rectas tangentes sucesivas:

xn+1 = xn -
f(xn)
f'(xn)

Es un algoritmo eficiente para encontrar aproximaciones de los ceros o raíces de una función real. Utiliza la primera derivada para trazar una recta tangente en el punto actual; la intersección de esta recta con el eje X se convierte en la siguiente aproximación mejorada. Su convergencia es cuadrática bajo condiciones ideales.

N-R Modificado (Raíces Múltiples)

Diseñado para conservar la convergencia rápida ante raíces múltiples incorporando la segunda derivada:

xn+1 = xn -
f(xn) · f'(xn)
[f'(xn)]2 - f(xn) · f''(xn)

Variante optimizada para funciones que presentan raíces múltiples (donde la curva es tangente al eje X). El método clásico pierde velocidad disminuyendo a convergencia lineal en estos casos. Al introducir la segunda derivada f''(x), esta modificación reestructura geométricamente el paso iterativo para mantener la velocidad de convergencia cuadrática original.
